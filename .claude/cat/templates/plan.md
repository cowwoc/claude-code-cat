# Plan Templates

Select the appropriate template based on work type.

---

## Feature Template

```markdown
# Plan: [Task Name]

## Goal
[What this feature accomplishes]

## Approach
[High-level implementation strategy]

## Files to Modify
- path/to/file1.ext - [reason]
- path/to/file2.ext - [reason]

## Dependencies
- [task-name] - [why needed]

## Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2

## Execution Steps
1. Step 1
2. Step 2
3. Step 3
```
